Literal Agenda is a minimalist personal agenda app. Your events are stored as plain text files and synced to a private GitHub repository you control.

Do you really need Google Calendar for your personal life? Dentist appointments, drinks with friends, a haircut next Tuesday — those don't need shared calendars, color-coded categories, or cloud tracking.

Literal Agenda is for your private schedule. Simple, readable, yours.

Features:
- 'Literal' Input: Type 415 for April 15, 930 for 9:30. No calendar scrolling.
- File-Based: Events are stored as plain .md files. No lock-in.
- GitHub Sync: Sync to a private GitHub repository for cross-device access.
- iCal Import: Import from Google Calendar, Apple Calendar, or any iCal-compatible app.
- Repeat: Daily, weekly, monthly, and yearly repeat support.
- Mini Calendar: Optional month view with event indicators.
- Customization: Font, size, accent color, and left-hand mode.

Events are stored in events/ and repeating/ directories in your repository as plain text files you can read, edit, or delete directly.

Literal Agenda is for people who want a quiet, private alternative to mainstream calendar apps.
